# Resume Analyser

AI-powered Resume Analyser that extracts skills and keywords from resumes using NLP.

## Tech Stack

Frontend:
- React.js

Backend:
- FastAPI
- Python

AI/NLP:
- Natural Language Processing
- Keyword extraction

## Features

- Upload resume
- Extract skills
- Analyze resume text
- Display extracted keywords

## Installation

Clone the repository

git clone https://github.com/dharshinivl26/resume-analyser.git

Install dependencies

pip install -r requirements.txt

Run backend

uvicorn main:app --reload

Run frontend

cd frontend
npm install
npm start
